## How to develop

```bash
# clone this repo or clone from your own fork
https://github.com/ekafyi/johnnydecimal-index-generator.git

# install dependencies (example with npm; you may also use yarn)
npm install

# run on develop mode
npm run start

# build to static "/build" directory
npm run build
```
